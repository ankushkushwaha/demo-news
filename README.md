## Overview

An iOS/iPad app built with Swift and SwiftUI. Users can search news, bookmark articles, and browse worldwide or their local area specific news.

## Features

- User can browse worldwide and local areas specific news articals.  
- Fetches news data from google new API  
- Allows users to bookmark news articals  
- Supports both iPhones and iPads  
- Full accessibility support:  
  - Dynamic text size  
  - VoiceOver  
  - Light and Dark Mode  
- Multi-language support, including:  
  - Left-to-right languages (e.g., English)  
  - Right-to-left languages (e.g., Hebrew)

## Technologies & Architecture

- Swift & SwiftUI  
- Clean MVVM architecture for modularity and testability  
- Unit tests with SwiftTesting  
- UI tests with XCUITest  
- Environment-specific configurations for development, staging, and production

## Compatibility

- Built with Xcode 26.0 on macOS Tahoe 26.0.1 
- Tested on iOS 26.0 and above  
- Works on both iPhones and iPads
